# OpenProcurement API Playground

Web console for OpenProcurement API


## Installation

### Using bootstrap

	$ python bootstrap.py
	$ bin/buildout -N
	$ flask run


### Using virtualenv

	$ virtualenv env
	$ . env/bin/activate
	$ pip install -r requirements.txt
	$ python setup.py install
	$ flask run
